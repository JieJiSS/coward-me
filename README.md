# Coward Me

A tool created for self-censoring.

> Refs:
>
> https://github.com/Muruwu/wordChecker (for api reference)
>
> https://github.com/morenjiujiu/sensitive_content_filter (word dictionary & algorithm)

## Install

```shell
$ git clone <this repo>
$ cd <directory name>
$ yarn install
```

## Usage

```shell
$ node ./censor.js "测试文本"
$ node ./anticensor.js "测试文本"
```
