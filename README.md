# Coward Me

A tool created for self-censoring.

> Refs:
>
> https://github.com/Muruwu/wordChecker (for api reference)
> https://github.com/morenjiujiu/sensitive\_content\_filter (word dictionary & algorithm)

## Install

```shell
$ git clone <this repo>
$ cd <directory name>
$ yarn install
```

## Usage

```shell
$ node ./index.js "测试文本"
```
